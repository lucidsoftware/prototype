Angular for Designers

We've put together a lot of code snippets to help designers use angularjs to make interactive prototypes. These are all done using angularjs and ng-animate. These examples help teach a basic understanding of how to build quick prototypes of an interactive interface.

The "Starter File" at the beginning of each example gives you the HTML and CSS you need to build the example. Follow the examples in order, and add the code to the starter file. Inside each codepen there are comments that draw your attention to code we are changing.

If you want to include lucid particle components, please see Starting in Codepen to add the right dependencies.
